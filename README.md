# Kammandor Intel

Real-time open-source intelligence for private capital — deals, counterparties, sanctions, markets and risk.

## Overview

Kammandor Intel is a live investment intelligence platform built for private capital professionals. It aggregates global flight tracking, maritime data, seismic monitoring, cyber threat feeds, news, and market intelligence into a single GPU-accelerated interface.

## Key Capabilities

| Domain | Description |
|--------|-------------|
| **Aviation** | Live commercial, private, and charter aircraft tracking |
| **Maritime** | Global port and chokepoint monitoring |
| **Seismic** | Real-time earthquake data (USGS) |
| **News** | 24/7 intelligence feed from global sources |
| **Cyber** | CVE and threat intelligence |
| **Markets** | Defence and commodity market tracking |
| **Sanctions** | OFAC SDN and entity screening |

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment Variables

| Variable | Description |
|----------|-------------|
| `UMAMI_WEBSITE_ID` | Umami analytics site ID |
| `NEXT_PUBLIC_SITE_TITLE` | Override default page title |

## Tech Stack

- Next.js 16
- MapLibre GL (WebGL map rendering)
- Gemini 2.0 Flash (AI analyst)
- TypeScript 5
